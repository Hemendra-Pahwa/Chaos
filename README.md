### A First-Person Shooter built in Unity Engine

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Overview

**Chaos** is a first-person shooter made in Unity, featuring dynamic ingame events every match with an immersive industrial environment.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Features

- First-person perspective
- Custom camera and aiming logic
- Enemy AI and combat encounters
- Different ingame modes

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Tools Used

| Unity : Engine where game runs
| Blender : 3D modeling & texture work

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Getting Started

### Prerequisites

- Unity Latest Version
- Git LFS (for large asset files)
