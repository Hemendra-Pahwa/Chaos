### A First-Person Shooter built in Unity Engine

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Overview

**Chaos** is a first-person shooter made in Unity, featuring dynamic ingame events every match with an immersive industrial environment.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Features

- First-person perspective
- Custom camera and aiming logic
- Enemy AI and combat encounters
- Different ingame modes

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Tools Used

| Unity : Engine where game runs
| Blender : 3D modeling & texture work

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Getting Started

### Prerequisites

- Unity Latest Version
- Git LFS (for large asset files)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/chaos.git

# Open in Unity Hub
# File → Open Project → select the cloned folder
```

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Project Structure

```
chaos/
├── Assets/
│   ├── Animations/       # Character & weapon animations
│   ├── Materials/        # Textures and shaders
│   ├── Models/           # 3D assets (.fbx, .glb)
│   ├── Prefabs/          # Reusable game objects
│   ├── Scenes/           # Unity scenes
│   └── Scripts/          # C# gameplay scripts
└── Packages/
```

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Controls

| `W A S D`: Move
| `Mouse` : Aim
| `Left Click` : Shoot
| `R` : Reloading
| `Space` : Jump
